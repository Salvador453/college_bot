Как использовать этот пакет:

1. Залей файлы на GitHub:
   - college_bot.py
   - requirements.txt

2. На Render.com создай новый Web Service, подключи репозиторий.

3. Build Command:
   pip install -r requirements.txt

4. Start Command:
   python college_bot.py

5. В файле college_bot.py замени ADMIN_IDS на свой Telegram ID.
